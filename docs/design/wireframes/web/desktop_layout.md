## レスポンシブ対応

### タブレット（中間サイズ）
- サイドバーを折りたたみ可能に
- グリッドレイアウトを2列→1列に変更
- フォントサイズを拡大

### モバイル互換モード
- ハンバーガーメニューでナビゲーション
- カード型レイアウトに変更
- タッチフレンドリーなボタンサイズ

## キーボードショートカット

- `Ctrl+N`: 新規作成
- `Ctrl+S`: 保存
- `Ctrl+F`: 検索
- `Ctrl+E`: 編集モード
- `Ctrl+Q`: クイズ開始
- `Tab`: フォーカス移動

## データ同期表示

- **リアルタイム同期**: モバイル⇔デスクトップ
- **同期状態アイコン**: ✅(完了) ⏳(進行中) ❌(エラー)
- **競合解決**: 最新更新を優先
- **オフライン対応**: ローカル保存→接続時同期
