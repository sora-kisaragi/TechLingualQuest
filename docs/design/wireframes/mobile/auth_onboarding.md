## 認証フロー・状態管理

### 認証状態の管理
```
未認証 → サインアップ/サインイン → メール認証 → オンボーディング → ダッシュボード
  ↑                                                                ↓
  └─────────────── ログアウト ←─────────────────────────────────────┘

エラー状態:
- ネットワークエラー
- 認証失敗
- バリデーションエラー
- サーバーエラー
```

### バリデーション

- **メールアドレス**: 形式チェック、重複確認
- **パスワード**: 8文字以上、英数字組み合わせ
- **名前**: 必須入力、文字数制限
- **リアルタイム検証**: 入力時即座にフィードバック

### セキュリティ機能

- **生体認証**: 指紋、Face ID（オプション）
- **自動ログアウト**: 一定時間後の自動ログアウト
- **セッション管理**: トークンの安全な保存
- **2FA準備**: 将来の2要素認証対応
